# 随机莲莲

随机的莲莲表情包或语音

## 使用

```
指令：
    莲莲/随机莲莲：发送莲莲表情包
    莲莲骂?[对象]：发送莲莲藏话
    狗叫 ?[长/短]：发送莲莲狗叫
    爱国：发送莲莲爱国表情
    罕见：发送莲莲罕见语音
    
    示例：随机莲莲
    示例：莲莲骂我
    示例：狗叫 长
```
如需使用莲莲表情包AI，请下载[预训练词向量](https://pan.baidu.com/s/1amvVC_Ue31KmBX0XVllMGA?pwd=j0q1)放在`\data`文件夹下
