# zhenxun_bot_plugins

真寻bot自用插件库，可以在真寻bot中使用，也可在nonebot2平台中通用

## 使用方法

下载对应的插件包，放在bot的插件文件夹`plugins/`下，重启bot即可加载插件并使用
